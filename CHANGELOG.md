# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-15

### Added
- Initial release
- Cloth vertex shader with height-based influence, wave animation, and velocity response
- Cloth fragment shader with edge detection and UV distortion
- ClothAnimator Lua module with spring physics and gust system
- Built-in presets: banner, flag, cape, curtain
- Configuration API for custom physics tuning
- Runtime methods: trigger_gust, set_gusts_enabled, set_spring
- Example project with banner, flag, and cape demos
- Complete documentation (README, Integration Guide, Parameters Reference, Troubleshooting)
